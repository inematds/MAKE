Mensageria Basica

Crie bots de Telegram, entenda WhatsApp Business e escolha o provedor certo. O inicio da automacao de atendimento.

6 topicos|Nivel: Iniciante|Pre-requisito: Modulo 1.8
1

Introducao ao Telegram Bot

O que e:

Telegram permite criar bots que interagem automaticamente com usuarios. API totalmente gratuita, poderosa e bem documentada - perfeita para aprender chatbots.

Por que aprender:

Telegram e muito popular no Brasil e tem a API mais acessivel do mercado. Perfeito para prototipar chatbots antes de ir para WhatsApp (mais complexo e caro).

Conceitos-chave:

BotFather, Bot token, Chat ID, Telegram API, Commands, Inline keyboards.

Criando um Bot

  1. Abra @BotFather no Telegram
  2. Envie /newbot
  3. Escolha nome e username
  4. Copie o token gerado
  5. Use no Make para conectar
2

Recebendo mensagens

O que e:

Configurar webhook no Make para receber mensagens enviadas ao bot. Processar texto, imagens, arquivos, audios e outros tipos de midia dos usuarios.

Por que aprender:

Recepcao de mensagens e a base de qualquer chatbot. Sem receber e processar input do usuario, impossivel criar interacao util.

Conceitos-chave:

Message update, Text message, Media message, User info, Chat type, Message ID.

Dados Recebidos

{
  "message": {
    "chat": {"id": 123456789},
    "from": {"first_name": "Maria"},
    "text": "Ola, preciso de ajuda!"
  }
}
3

Enviando respostas automaticas

O que e:

Usar modulo Telegram no Make para enviar mensagens de volta ao usuario. Pode ser texto simples, imagens, documentos, ou mensagens com botoes interativos.

Por que aprender:

Resposta automatica e a funcao primaria de qualquer bot. E onde a magica da automacao de atendimento realmente acontece.

Conceitos-chave:

Send message, Reply markup, Inline keyboard, Parse mode (HTML/Markdown), Photo/Document.

4

Trabalhando com grupos

O que e:

Bots podem participar de grupos e canais do Telegram. Comportamento diferente: precisa ser mencionado ou ter permissao de admin para ver todas mensagens.

Por que aprender:

Grupos multiplicam o alcance do bot. Um unico bot pode atender centenas de usuarios simultaneamente em grupos de comunidade.

Conceitos-chave:

Group chat, Channel, Admin rights, Privacy mode, Mentions, Group ID vs Chat ID.

Importante

Em grupos, bots so recebem mensagens quando sao mencionados (@seubot) ou quando tem "Privacy mode" desativado nas configuracoes do BotFather.

5

Introducao ao WhatsApp

O que e:

WhatsApp Business API permite automacoes similares ao Telegram, porem com custos, regras mais rigidas e complexidade maior. E o canal dominante no Brasil.

Por que aprender:

WhatsApp e onde 99% dos clientes brasileiros estao. Automacao de WhatsApp tem o maior impacto comercial real no Brasil.

Conceitos-chave:

WhatsApp Business API, Message templates, Session messages, 24h window, Opt-in.

WhatsApp vs Telegram

Telegram:

Gratis, facil, menos usuarios

WhatsApp:

Pago, complexo, mais usuarios

6

Escolhendo provedor de WhatsApp

O que e:

Comparar opcoes de API WhatsApp: Z-API, Evolution API, Twilio, Meta Business API. Cada um com custos, recursos e complexidades diferentes.

Por que aprender:

Escolha errada custa caro ou limita funcionalidades. Entender as opcoes evita retrabalho e surpresas desagradaveis no futuro.

Conceitos-chave:

Z-API, Evolution API, Twilio, Official API, Unofficial API, Pricing models.

Comparativo de Provedores

Z-APIR$49+/mes | Facil | Nao-oficial
Evolution APISelf-hosted | Gratuito | Nao-oficial
Twilio$0.005/msg | Oficial | Complexo
Meta Cloud1000 gratis | Oficial | Templates

Recomendacao para Iniciantes

Comece com Z-API pela facilidade de setup. Migre para Evolution (self-hosted) quando escalar, ou Meta Cloud API para casos que exigem API oficial.
